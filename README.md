# seckill（Java高并发秒杀API）

#### 所用技术
 - spring
 - springMVC: MVC框架
 - Tomcat: web容器
 - mybatis: ORM框架
 - bootstrap: css/html框架
 - JQuery: JS框架
 - Redis: NOSQL数据库
 - MySQL: 关系型数据库
 - Logback: 日志框架
 - JUnit: 单元测试
